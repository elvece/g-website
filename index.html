The Trail

Service to generate a personalized hike recommendation in Colorado.

Javascript  JQuery  Node/Express

The Trail Description

This application seeks to solve the often difficult decision of which hike to choose in colorful Colorado. Users complete a short questionnaire from which they receive an individualized hike recommendation. This project is a current work in progress, as I hope to expand the functionality to include hike data visualizations, an interactive trail map, user contributed images and comments, and a weather forecast.
